Paste Mode

Ctrl + V is used to enter the Paste Mode.
To activate Paste Mode, go through the following sequence -
  • Press Ctrl + V
  • Keep Ctrl pressed
  • Release V
While being in paste mode ( Ctrl pressed ), following are the actions available - 
V - Move forward through Multiple Clipboards (from 1 to 2 , 2 to 3, …)
C - Move backward through Multiple Clipboards (from 3 to 2 , 2 to 1, …)
S - Add current Clip to Windows Clipboard
X - Toggle between Cancel, Delete, Copy, Move and Delete All MODE 
E - Export current clip
A - Jump to first clip
F - Activate Search / Entering Search Mode
H - Edit Clip
Z - Toggle pasting formats
T - Add tags to clips
↵ (i.e. Enter) - Start Multiple Pasting Session
  (i.e. Space) - Fix/un-fix a clipboard at its position (Fixate)
↑ (i.e. Up) - Change channel to 1 up (+1)
↓ (i.e. Down) - Change channel to 1 down (-1)
1..9 - Move to clip {num} ahead/behind of current clip
F1 - View listing of all keys in Paste Mode
See this ClipjumpCustom.ini example to change these keys

Search in Paste Mode

The following key sequence enters Search Mode while being in Paste Mode:
  • While Ctrl pressed ...
  • Press F
  • Release Ctrl and F
Now you are in Search Mode within Paste Mode and several commands are available via keypress: 
Enter - Paste
Home / Esc - Cancel
↑ (i.e. Up) - Move one clip up (+1)
↓ (i.e. Down) -Move one clip down (-1)
See ClipjumpCustom.ini to change these keys

Cancel, Delete, Copy, Move and Delete All Mode

While being in Paste Mode, pressing X repeatedly toogles between several modes: 
X - Cancel Mode - cancel the current paste operation
X - X - Delete Mode - delete current clip
X - X - X - Move Mode - Move clip to a different channel
X - X - X - X - Copy Mode - copy clip to a different channel
X - X - X - X - X - Delete All Mode - deletes all clips

Action Mode

You can find here a few more details about Action Mode
Ctrl + Shift + A is used to enter the Action Mode. A dialog box pops up, offering a lot more "actions" which can be performed. These "actions" can be accessed via keypress: 
C - Copy File path(s)
D - Disable Clipjump
E - Settings
F - Copy File Data
F1 - Help
F2 - Paste Mode Shortcuts
H - Clipboard history (also accessible via direct shortcut Win + C)
L - Ignore Windows Manager
M - Plugin Manager
0 - Channel Organizer
P - PitSwap
S - Select Channel
T - One Time Stop
X - Copy active folder path
Esc - Exit Action Mode Window
